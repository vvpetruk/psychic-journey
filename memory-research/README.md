# Memory Research Workspace

Memory Research Workspace is a canonical architecture workspace that integrates:
- Titans-style neural long-term memory and deployable memory-centric model variants
- Nested Learning ideas including multi-timescale updates, continuum memory, and optimizer-memory coupling

## Current strategy
- Treat `titans-pytorch-mlx` as the primary implementation base
- Treat `nested_learning` as the conceptual and mechanism reference
- Build one coherent architecture line that is practical, extensible, and deployment-oriented

## Scaffold goals
- unify memory abstractions
- experiment with multi-timescale memory updates
- explore nested-learning-informed Titans derivatives
- converge toward one main deployable repo
